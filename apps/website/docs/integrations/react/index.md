# Farfetched and React

Integration is distributed as a separate package, you have to install it before usage

with `pnpm`

```sh
pnpm install @farfecthed/react
```

with `yarn`

```sh
yarn add @farfecthed/react
```

with `npm`

```sh
npm install @farfecthed/react
```

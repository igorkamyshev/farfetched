# Farfetched and Solid

Integration is distributed as a separate package, you have to install it before usage

with `pnpm`

```sh
pnpm install @farfecthed/solid
```

with `yarn`

```sh
yarn add @farfecthed/solid
```

with `npm`

```sh
npm install @farfecthed/solid
```
